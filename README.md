# CSV Birleştirici (Streamlit)

## Lokal
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud
- GitHub repo'ya `app.py` ve `requirements.txt` yukle
- Streamlit Cloud -> New App -> app.py

## Notlar
- Varsayilan: ilk dosyanin basligini kullanir, digerlerinin basligini kontrol eder.
- Strict header acik olursa baslik farkinda islemi durdurur.
- Cikti ZIP: birlesik CSV + rapor.
